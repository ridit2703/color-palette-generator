# 🎨 Color Palette Generator

A simple web app that generates random color palettes in HEX or RGB format.

## 🚀 Features

* Generate random colors
* Choose between HEX and RGB formats
* Filter light and dark tones

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 🔧 How to Use

1. Select format (HEX/RGB)
2. Select tone (Light/Dark/All)
3. Click "Generate"

## 📂 Project Setup

Just open `index.html` in your browser.
